### Tentang Aplikasi
Aplikasi ini dirancang khusus untuk memetakan dinamika kelembagaan, tingkat pengaruh, potensi kolaborasi, serta gesekan/konflik antar-aktor kepentingan (*stakeholders*) dalam suatu ekosistem kebijakan atau program strategis secara interaktif.

### Fitur Utama:
* **Manajemen Aktor Dinamis:** Menambah daftar aktor secara *real-time*.
* **Matriks Atribut Profil:** Pendataan karakteristik dasar kelembagaan.
* **Matriks Relasional Interaktif:** Input skor bernilai menggunakan *dropdown selection* untuk relasi *Collaboration*, *Influence*, *Conflict*, dan *Power*.
